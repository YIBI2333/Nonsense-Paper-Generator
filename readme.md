# 胡说八道论文生成器 Nonsense-Paper-Generator

一本正经地胡说八道——用标准学术格式生成完全放飞的混沌论文



## **该项目仅供娱乐，请勿在正规用途使用！**



## 快速开始



使用 "index.html"或使用"paper_generator.py"。

## 特点



- 参考 IEE/APA 学术论文格式
- 包含虚构的图表、数据表格和公式
- 参考文献纯属虚构
- 生成内容纯属虚构
- HTML

## 使用说明



1. 打开页面输入论文关键词
2. 点击“重新生成”按钮重新生成获取论文
3. 支持文档导出论文
